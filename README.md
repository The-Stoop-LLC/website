# THE STOOP — Agency Website

Digital advertising agency specializing in influencer marketing and creator coaching.

**Live Site:** [thestooppgh.com](https://thestooppgh.com) *(update with your actual domain)*

## Tech Stack
- Pure HTML/CSS/JS — no build step required
- Google Fonts (Outfit, Anton)
- CSS custom properties for theming
- Intersection Observer for scroll animations

## Brand
- **Colors:** True Black, True White, Tangerine Yellow (#FFCC00)
- **Fonts:** Lemon Milk (→ Outfit), Akira Expanded (→ Anton), Helvetica
- **Aesthetic:** Minimalistic, monochrome, black + yellow

## Deploy
This is a single `index.html` file. Deploy anywhere:
- **GitHub Pages:** Push to `main`, enable Pages in repo settings
- **Netlify/Vercel:** Drag and drop the folder
- **Custom domain:** Point DNS to your hosting provider
